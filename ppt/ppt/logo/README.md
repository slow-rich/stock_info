# Slow Rich — Logo 設計風格記錄

## Logo 1：Turtle Golden（系列）
四個版本迭代，均為六角龜殼 + 金色折線。

### _1：原始版
**檔案：** `logo_1_turtle-golden_1.html`

| 項目 | 內容 |
|---|---|
| 圖標 | 龜殼六邊形（3 條對角分隔線 + 中央小六邊形）+ 複利成長平滑曲線 |
| 主色 | 深森林綠 `#1B4332` |
| 輔色 | 中綠 `#2D6A4F`（分隔線）、香檳金 `#D4AF37` |
| 線條 | 平滑貝茲曲線（左下 → 右上）|
| 氛圍 | 專業、永續、穩健財富 |

---

### _2：股市折線（規律）
**檔案：** `logo_1_turtle-golden_2.html`

| 項目 | 內容 |
|---|---|
| 圖標 | 龜殼六邊形 + 規律鋸齒折線（等幅震盪）|
| 主色 | 中森林綠 `#2D6A4F`（較 _1 亮）|
| 輔色 | 薄荷綠 `#52B788`（分隔線）、香檳金 `#D4AF37` |
| 線條 | 13 個節點等幅鋸齒，轉折點有小金點 |
| 備註 | 振幅過於規律，已被 _3 取代 |

---

### _3：股市折線（不規律）⭐ 目前使用版
**檔案：** `logo_1_turtle-golden_3.html`

| 項目 | 內容 |
|---|---|
| 圖標 | 龜殼六邊形 + 劇烈不規律折線（模擬真實股市）|
| 主色 | 中森林綠 `#2D6A4F` |
| 輔色 | 薄荷綠 `#52B788`（分隔線）、香檳金 `#D4AF37` |
| 線條 | 8 個轉折點，振幅 12–33（不規律），含早期大幅回撤後強力反彈 |
| 折線特徵 | 上漲 19 → 回撤 17（幾乎打回原點）→ 強反彈 28 → 急漲 33 → 最終衝頂 |
| 填色 | 線下方淡金色底面（opacity 0.07）|
| 轉折點 | 8 個中間轉折點有小金點（r=1.8），終點大金點（r=4.5）|

### _4：海軍藍版（台灣市場適用）⭐ ep01 目前使用
**檔案：** `logo_1_turtle-golden_4.html`

| 項目 | 內容 |
|---|---|
| 圖標 | 同 _3（8 個轉折不規律折線），顏色改為藍色系 |
| 主色 | 海軍藍 `#1B3A6B` |
| 輔色 | 皇家藍 `#2E5FA3`（分隔線）、淡藍 `#4A7FD4`（內六邊形邊框）、香檳金 `#D4AF37` |
| 改色原因 | 台灣股市慣例：綠色代表下跌，投資教育品牌不宜使用綠色 |
| 品牌搭配 | 海軍藍 + 香檳金 = 頂級金融品牌配色（Goldman Sachs、JP Morgan 風格）|

## 技術規格（Turtle Golden 系列共用）

| 項目 | 規格 |
|---|---|
| ViewBox | `0 0 100 100` |
| 外六邊形 | 中心 (50,50)，半徑 44，頂點朝上（起始角 -90°）|
| 內六邊形 | 中心 (50,50)，半徑 18 |
| 龜殼分隔線 | 連接對面頂點：[0,3]、[1,4]、[2,5] |
| 折線起點 | (14, 82) 左下，終點 (85, 15) 右上 |
| ClipPath | 折線裁切在外六邊形內 |
| 技術 | React + Babel（in-browser），無需建置工具 |

## 使用建議

| 場景 | 建議 Logo |
|---|---|
| 投影片封面（台灣受眾） | Turtle Golden _4（海軍藍）|
| 投影片頁首 / 頁尾浮水印 | Turtle Golden _4，建議 40–52 px |
| 深色背景投影片封面 | Turtle Golden _4（深色背景效果最佳）|
| 國際受眾 / 不在乎顏色慣例 | Turtle Golden _3（綠色版）|
| 正式徽章 / 頻道 Logo | Badge Navy |
| 個人品牌 / 放鬆風格 | Spiral Earth |
| Keynote / Figma 貼入 | 複製各檔案內的 `<svg>` 原始碼，天生透明背景 |
