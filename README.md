1: 台股(0050)，vt，s&p500 : 年化報酬率
2: 雖然年化報酬率是 8%~12%，但從表格來看，介於8%~12%的年份，其實比例不高，常常出現 極端值
3: 挑選個股，它的年化報酬率 要超過 市值型ETF(例如：0050，vt，s&p500)，機率大約只有 10~20% (圖表 要怎麼產生)
4: 不要挑選個股，直接買市值型ETF(例如：0050，vt，s&p500), 勝率最高，報酬率勝過挑選個股
5: 透過 股債平衡，降低整體本金的波動
這邊的股票，是指 市值型ETF(例如：0050，vt，s&p500)
債卷是指 短天期債卷(例如：00719b, BSV): 股市大跌時，短天期債卷 波動較小
